I will show you how to create a fantasy themed house model, how to colour it, how to set the lights and how to render it correctly. I was a bit pissed off as you might noticed in my voice, so I made some mistakes. To name them, I did not delete all the hidden faces, and you should always delete hidden faces (its OK if you are going to destroy your model, explode it or something).

Edit mode:
E-> extrude
R-> rotate
S-> size
G-> move
X-> delete
Ctrl+R-> edge loop
P-> separate
XYZ-> axis
Shift+S -> cursor to selected etc
alt+M-> merge
L-> select linked
Shift+B -> box select
Shift+A -> add object
Ctrl+Z -> undo
Shift+D -> duplicate
Alt+RMB -> select edge loop
Alt+A -> start animation
H -> hide
Alt+H -> unhide
Shift+H -> hide everything except selection
