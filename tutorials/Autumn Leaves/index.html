In this video, I will show you, how to set up a particle system for a simple plane emitting autumn leaves from itself.

The simulation is not perfect of course, the leaves could be blown by an actual wind force field, making the flying effect more believable, but I ran out of time, and I am not a big fan of the particle system anyway. The final result is not photo realistic, but its all good enough to teach you some basic things about particles, using alpha textures and using groups for the particle system.

That's enough about it, lets start:

Summary:

Just add a plane, set the UV layout, then give it a colour texture and an alpha map. Set the material to transparent, without reflecting any specularity, and set the alpha maps alpha value to -1 and tick in the calculate box, and make sure to disable the colour value for it. Make some leaves, group them and set the particle system to pick randomly from that group. Make all settings as random as you see beneficial, then make the start to -50 to start the simulation before your render starts, and end to 100. Set the lifetime to 150 so your leaves wont disappear. Make a nice little wallpaper, add some defocus, and you are ready. After that, set up your renders file output, AVI MPEG-4 (DivX) and render the animation.
